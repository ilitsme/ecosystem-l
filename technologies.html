Направление «НИИИН Технологии» отвечает за разработку аппаратных и программных решений, обеспечивающих функционирование экосистемы неразрушающего контроля (НК):

Единый протокол обмена данными высокого уровня для всех Smart-устройств

Регламентированный атрибутивный состав данных, соответствие которому позволит использовать их в рамках единой цифровой инфраструктуры. Передача данных по протоколу позволит обеспечить прослеживаемость измерений, подготовит результаты для последующий верификации и анализу.

smart-оборудование

Реализующее передачу данных по протоколу. Его главное отличие от стандартного дефектоскопа заключается в наличии возможности по сбору и передаче данных НК их в центр обработки данных.

фильтр для верификации данных НК

Размещение на входе в центр обработки данных программного фильтра, который будет обеспечивать сравнение поступающих данных с требованиями нормативно-технической документации, позволит обеспечить максимальную объективность данных НК.

сервисы по обработке упорядоченных, верифицированных данных НК

Позволяющие применить технологии машинного обучения и аналитические модели для использования потенциала данных НК с целью:

определения путей совершенствования технологии производства работ

изменения режимов эксплуатации по критерию реального технического состояния и требуемого времени эксплуатации

проведения экспресс-анализа качества работ подрядчиков (производственных участков)

внедрения подходов предиктивной аналитики с целью определения остаточного жизненного цикла и риска эксплуатации изделий и их составных частей

выявления нежелательных тенденций во взаимоотношениях отдельных элементов производственной цепочки

определения участков производства работ по изготовлению продукции (строительству сооружений), требующих немедленного воздействияs

определения технического состояния объектов на основании данных, поступающих от нескольких методов НК, установление сроков и методики последующего технического диагностирования отдельных элементов объектов

Визуализация данных НК позволит определить узловые точки для принятия управленческих решений, а поддержка со стороны ИИ позволит сконцентрировать внимание на действительно важном.

Для того, чтобы решения «НИИИН Технологии» можно было внедрять на федеральном уровне, специалисты активно участвуют в деятельности ТК 371 «Неразрушающий контроль». Для повышения эффективности взаимодействия участников разработки и сокращения сроков рассмотрения проектов национальных стандартов институтом создана цифровая платформа, в рамках которой проводится обмен мнениями и голосование членов рабочей группы.

Текущие разработки направления «НИИИН Технологии»

На данный момент активное развитие получили следующие проекты:

№1

Smart-оборудование

В 2023 году разработано Техническое задание на создание программно-аппаратного комплекса на основе отечественной микроэлектронной компонентной и программной базы для модернизации дефектоскопа УГТ.

TapiRUS_ассистент

Действующее Android-приложение, позволяющее выполнять косвенные измерения с использованием универсального шаблона специалиста НК TapiRUS, а также предназначенное для определения норм отбраковки при визуальном и измерительном контроле.

№3

Компонента «Вихреток»

Программное обеспечение для взаимодействия с Smart-устройствами НК для получения объективных данных в режиме реального времени. В функционал ПО входит: визуализация данных, управление параметрами контроля, формирование протоколов и дефектограмм, обеспечение последующей передачи данных в ЦОД. Является составной частью программно-аппаратного комплекса для выполнения задач сбора, обработки и анализа больших данных.

Также, сформировано понимание концепции универсального протокола обмена данных для различных источников (оборудование, реализующее различные технологии НК, аттестационные центры, метрологические службы)
